## Описание

Приложение позволяет работать с таблицами и данными через консоль:
- создавать и удалять таблицы,
- добавлять, читать, обновлять и удалять записи,
- хранить данные в JSON-файлах.

Проект реализован с использованием декораторов, замыканий и обработки исключений.

## Управление БД

Доступные команды:
- create_table <имя> <столбец:тип> ... (создает новую таблицы)
- list_tables (показывает список всех таблиц в базе)
- drop_table <имя> (удаляет таблицу)
- info <имя> (показывает: имя таблицы, список столбцов, количество записей)
- insert into <таблица> values (...) (добавляет новую запись в таблицу (ID генерируется автоматически))
- select from <таблица> (показывает все записи таблицы)
- select from <таблица> where <столбец> = <значение> (показывает записи по условию)
- update <таблица> set <столбец> = <значение> where <столбец> = <значение> (обновляет записи по условию)
- delete from <таблица> where <столбец> = <значение> (удаляет записи по условию)
- help (показывает справку по командам)
- exit (выход из программы)

## Декораторы и замыкания

В проекте реализованы:
- handle_db_errors — централизованная обработка ошибок
- confirm_action — подтверждение опасных операций
- log_time — замер времени выполнения функций
- кэширование select с использованием замыканий
